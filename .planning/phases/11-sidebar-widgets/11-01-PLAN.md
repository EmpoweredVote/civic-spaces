  <action>
1. Run `npm install recharts` to add recharts to the project.

2. Create `src/types/compass.ts` with TypeScript types:
   - `CompassCategory`: `{ id: string; title: string; topics: Array<{ id: string; title: string; short_title: string | null; question_text: string }> }`
   - `CompassAnswer`: `{ topic_id: string; value: number; write_in_text: string | null; visibility: string; inverted: boolean; updated_at: string }`

3. Create `src/types/representatives.ts` with TypeScript types:
   - `PoliticianFlatRecord`: `{ id: string; full_name: string; office_title: string; photo_origin_url: string; district_type: string; government_type: string; is_vacant: boolean; images: Array<{ id: string; url: string; type: string; photo_license: string; focal_point: string | null }> }`
   - NOTE: The `party` field exists in the API response but MUST NOT be included in this type. Omitting it from the type prevents accidental rendering (anti-partisan policy).
   - Export `BRANCH_ORDER` constant: `Record<string, number>` mapping district_type values to sort order (NATIONAL_EXEC: 0, STATE_EXEC: 1, LOCAL_EXEC: 2, NATIONAL_UPPER: 10, NATIONAL_LOWER: 11, STATE_UPPER: 12, STATE_LOWER: 13, COUNTY: 14, LOCAL: 15, SCHOOL: 16, NATIONAL_JUDICIAL: 20, JUDICIAL: 21).
   - Export `getRepPhoto(rep: PoliticianFlatRecord): string | null` — returns `rep.images[0]?.url ?? (rep.photo_origin_url || null)`.

4. Create `src/hooks/useCompassData.ts`:
   - Uses `@tanstack/react-query` `useQuery` for both calls.
   - `fetchCompassCategories()`: GET `https://accounts.empowered.vote/api/compass/categories` (no auth). Returns `CompassCategory[]`.
   - `fetchCompassAnswers(token)`: GET `https://accounts.empowered.vote/api/compass/answers` with Bearer token from `localStorage.getItem('cs_token')`. Returns `CompassAnswer[]`. If no token, return `[]`.
   - `useCompassData(userId: string | null)` returns `{ categories, answers, isLoading, isUncalibrated }`. `isUncalibrated` = `!userId || answers.length < 3`.
   - `staleTime` for categories: 30 min. For answers: 5 min.
   - Export `buildChartData(categories, answers)` function: maps each category to `{ category: cat.title, value: average of answered topic values }`, filters out categories with value 0.

5. Create `src/hooks/useRepresentatives.ts`:
   - `fetchRepresentatives(token)`: GET `https://accounts.empowered.vote/api/essentials/representatives/me` with Bearer token. CRITICAL: check `response.status === 204` before calling `.json()` — return `[]` for 204. Returns `PoliticianFlatRecord[]`.
   - `useRepresentatives(userId: string | null)` returns the React Query result. `enabled: !!userId && !!token`. `staleTime: 10 * 60 * 1000`.
  </action>

    - Grep for `useCompassData` and `useRepresentatives` exports in hooks directory
