1. Downloads 3 public domain photos from their source URLs to a local `tmp/` directory

3. Logs the confirmed CDN URLs

**Photo sources (public domain):**
- **White House**: Use a wide landscape photo from Wikimedia Commons. Search for "White House Washington DC" on commons.wikimedia.org. Get the DIRECT file URL (must start with `https://upload.wikimedia.org/` and end in `.jpg`). Do NOT use the wiki page URL. Aim for a wide/landscape orientation suitable for 16:5 aspect ratio cropping.
- **Indiana State Capitol**: Search Wikimedia Commons for "Indiana Statehouse" or "Indiana State Capitol Indianapolis". Same rules — direct upload.wikimedia.org URL only.
- **Monroe County Courthouse Bloomington Indiana**: Search Wikimedia Commons. Select a horizontal/landscape photo. Direct URL only.

**Upload paths in bucket:**
- `bloomington/white-house.jpg`
- `bloomington/indiana-capitol.jpg`
- `bloomington/courthouse.jpg`

**Script structure:**
```typescript
import { createClient } from '@supabase/supabase-js'

// Requires SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY env vars
const supabase = createClient(
  process.env.SUPABASE_URL!,
  process.env.SUPABASE_SERVICE_ROLE_KEY!
)

const PHOTOS = [
  { sourceUrl: '...wikimedia direct URL...', storagePath: 'bloomington/white-house.jpg' },
  { sourceUrl: '...wikimedia direct URL...', storagePath: 'bloomington/indiana-capitol.jpg' },
  { sourceUrl: '...wikimedia direct URL...', storagePath: 'bloomington/courthouse.jpg' },
]

async function main() {
  for (const photo of PHOTOS) {
    // fetch source URL -> get ArrayBuffer -> upload to Storage
    const response = await fetch(photo.sourceUrl)
    if (!response.ok) throw new Error(`Failed to download ${photo.sourceUrl}: ${response.status}`)
    const buffer = Buffer.from(await response.arrayBuffer())

    const { error } = await supabase.storage
      .from('slice-photos')
      .upload(photo.storagePath, buffer, {
        contentType: 'image/jpeg',
        upsert: true,
        cacheControl: '31536000', // 1 year
      })
    if (error) throw error

    const { data } = supabase.storage.from('slice-photos').getPublicUrl(photo.storagePath)
    console.log(`Uploaded: ${data.publicUrl}`)
  }
}

main().catch(console.error)
```

**Run the script:**
```bash
SUPABASE_URL=... SUPABASE_SERVICE_ROLE_KEY=... npx tsx scripts/upload-bloomington-photos.ts
```

**After running:** Verify each CDN URL returns 200 by curling them. Save the 3 confirmed CDN URLs — they are needed for Task 2.

**IMPORTANT pitfall avoidance:**
- Do NOT download the Wikimedia *page* URL (commons.wikimedia.org/wiki/File:...). You need the *direct image* URL (upload.wikimedia.org/wikipedia/commons/...).
- If a photo is very large (10MB+), that is fine — Supabase Storage handles it and the bucket allows up to 5MiB. If a source image exceeds 5MiB, resize it or find an alternative smaller source.
- Do NOT commit photo binaries to git. The script downloads them transiently.
