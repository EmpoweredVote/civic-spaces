Import `HeroBanner` from `./HeroBanner` at the top of AppShell.tsx.

Inside the feed column (the left column of the two-column grid from Plan 01), add the HeroBanner ABOVE the feed tab panels but INSIDE the feed column div. The hero banner should be outside the CSS-hidden tab divs so it is always visible and swaps based on `activeTab`.

Compute the active slice data from `slices[activeTab as SliceType]` and pass it to HeroBanner:

```tsx
{/* Feed column */}
<div className="flex flex-col overflow-hidden min-h-0">
  {/* Hero banner — shows data for the active tab's slice */}
  {(() => {
    const activeSlice = slices[activeTab as SliceType]
    if (!activeSlice) return null
    return (
      <HeroBanner
        sliceType={activeSlice.sliceType}
        sliceName={TAB_LABELS[activeTab]}
        geoid={activeSlice.geoid}
        memberCount={activeSlice.memberCount}
        siblingIndex={activeSlice.siblingIndex}
      />
    )
  })()}

  {/* Feed tab panels (unchanged from Plan 01) */}
  {FEED_TABS.map((tabKey) => {
    // ... existing feed panel code
  })}
  {/* ... volunteer panel ... */}
</div>
```

**Important:** The HeroBanner must NOT be inside a `flex-1` container — it should take its natural height (determined by aspect ratio). The feed panels below it get `flex-1` to fill remaining space. The hero banner does NOT scroll with the feed; it is fixed at the top of the feed column and the feed scrolls independently below it.

To achieve this layout:
- HeroBanner: no flex-1, natural height from aspect ratio
- Feed panel wrapper: `flex-1 overflow-hidden min-h-0` (takes remaining space)

This means the feed column structure is:
```
<div className="flex flex-col overflow-hidden min-h-0">
  <HeroBanner ... />           <!-- natural height -->
  <div className="flex-1 ..."> <!-- fills remaining -->
    <SliceFeedPanel ... />
  </div>
</div>
```

Adjust the existing feed tab panel wrappers if needed — the active panel's outer div should have `flex flex-col flex-1 overflow-hidden min-h-0`.

Do NOT add `import type { SliceType }` if it's already imported. Check existing imports first.
